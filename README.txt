Langages : Javascript ( Librairies p5.js, dom.js ), Html
Interface :
    L'interface est composée de :
        - un lien vers un solveur en ligne
        - une grille
        - des boutons
            - Générer : génère une grille
            - Résoudre : génère un fichier contentant la forme normale conjonctive représentant la grille
            - Reset : nettoie la grille
        - une checkbox pour choisir la couleur finale souhaitée
        - un slider pour définir la taille de la grille



Etapes pour résoudre une grille :
    1) Choisir la couleur cible grâce à la checkbox
    2) Nettoyer la grille courante avec le bouton Reset (optionel)
    3) Générer une grille avec le bouton Générer
    4) Copier la forme normale conjonctive générer via le bouton Résoudre
    5) Se rendre sur le lien cliquable en haut de la page 'solveur en ligne' ( https://msoos.github.io/cryptominisat_web/ )
    6) Copier le contenu du fichier dans la partie haute de la page web ( en effacant son contenu au préalable )
    7) Cliquer sur la flèche en haut à droite pour résoudre la forme normale conjonctive
    8) Tout en bas de la partie basse de la page, les solutions apparaîssent derrière un V
    9) Retourner sur la grille et cliquer sur une case seulement si le solveur n'a pas placé de '-' devant son numéro dans la solution
